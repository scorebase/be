<h1  align="center">Scorebase API Service</h1>

  

## How to run

* Clone repo

* Run

```bash
	npm install
```

* Set up your .env file with the .env.example template

  

## Running the app

```bash
	npm run dev
```

  

## Tests
#### Unit tests
```bash
	npm run test:unit
```
#### Integration tests
```bash
	npm run test:integration
```
#### All tests
```bash
	npm run test:all
```

## Linting
#### Lint
```bash
	npm run lint
```
#### Lint and fix
```bash
	npm run lint:fix
```
